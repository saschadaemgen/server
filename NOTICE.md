# Third-Party Notices

unifix-server bundles or remote-loads the following third-party
assets. The unifix source code itself remains proprietary; this
file acknowledges upstream contributions per their licenses.

## Claude-Design Frontend Library

The complete tenant intercom-viewer UI and the admin shell
(tokens.css, components.css, interactions.js, plus the HTML
snippets in `server/internal/httpserver/design-library/`) are
generated by Anthropic's Claude-Design tooling for unifix as a
commissioned design system.

- Origin:    Claude-Design (Anthropic), commissioned for unifix
- License:   internal use within unifix-server, terms agreed
             between Sascha Daemgen and Anthropic
- Includes:  inline SVG icons drawn by Claude-Design (no
             external icon library), gradients, animations, and
             component classes

The library is treated as approved spec; integration code may
adapt it (form action paths, field names) but must not change
CSS class names, token values, or the inline SVG icon paths.

## Tailwind CSS

Tailwind CSS is no longer linked in any served page after
S13-02-FIX3 (the Claude-Design library replaced all utility
classes with semantic component classes). This entry remains
for archival reference; the dependency can be removed entirely
once no future template re-introduces it.

- Project: https://github.com/tailwindlabs/tailwindcss
