# unifix-server

Internal Go monorepo for the unifix platform.

## Modules

- server: main product server (pool manager, REST API, UI)
- mock: UA Intercom Viewer simulator for testing
- license-server: cloud component for license validation
- shared: common packages (types, proto, logging)
